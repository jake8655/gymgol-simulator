A small game as a school project
![gymgol-simulator](https://github.com/user-attachments/assets/dd649393-5ccf-4ae8-b727-c399dd81ffe3)

## How to play
1. Install [python](https://www.python.org/downloads/)
2. Install pygame:
```
pip install pygame
```
3. Run the game
```
python main.py
```
